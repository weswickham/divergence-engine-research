## Researcher's Introduction

*Reproduced from The_Last_Library_human_and_AI_compare.docx, authored by Wes Wickham, September 2025.*

The following results are an attempt to establish whether LLMs can successfully be used as an augment or replacement for brainstorming in the creative process. This represents a first attempt to gain a broad overview and understanding of the prompts needed to generate suitably appropriate responses.

The test was conducted using the current models of ChatGPT-5 and Claude Sonnet 4 using custom instruction sets on Sept 20–21, 2025. The custom instructions were established after a period of iterative prompt development and testing. The instruction set entitled 'The Divergence Engine' is an ongoing project.

The project brief or design challenge established for test was to begin brainstorming idea sparks for a book cover for a hypothetical dystopian story called 'The Last Library' (a name suggested by Claude). The test shows the raw outputs from my own responses in a 30-minute timed exercise and the corresponding LLM outputs.

The exact prompt instructions and fine-tuning for each LLM output is recorded. Below are the final outputs.
